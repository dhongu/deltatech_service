- Features:

  - Gestionare timp de deplasare catre lucrare
  - La apasarea butonului de start se va face automat determinarea locatiiei (lat,lng) si se compara cu locatia de destinatie
  - Daca nu sunt in apropiere se va considera ca se incepe deplasarea
  - La apasarea butonului de stop se va face iarasi determinarea locatiei si se va calcula distanta parcursa si timpul de deplasare
  - Coordonetele se vor salva in chaterele de activitate
  - Timpul si durata de deplasare se vor adauga la comadna de vanzare
